# Flowra — Executive White Paper

> **Classification:** Confidential | **Version:** 1.0 | **Date:** April 2026  

---

## Executive Summary

**Flowra** is a mobile-first personal state reconstruction platform. It transforms fragmented daily activity — text, files, calendar events, emails — into continuous, AI-powered awareness of what's happening in a user's life.

Unlike tools that require users to *organize*, Flowra requires them only to *dump*. The system extracts action items, tracks deadlines, detects blockers, and surfaces a real-time "life state" dashboard.

> *"I dump things here and it tells me what's going on."*

---

## 1. The Problem

Knowledge workers use **7–12 productivity tools** daily. Yet:
- **68%** forget follow-ups within 48 hours
- **73%** feel overwhelmed by their own systems
- **81%** have abandoned a productivity app in the past year

**The paradox:** Tools designed to reduce cognitive load are themselves a source of it.

### Why Existing Solutions Fail

| Category | Examples | Failure |
|---|---|---|
| **Organization-first** | Notion, Obsidian | Requires structure upfront |
| **Task managers** | Todoist, Asana | No context, no state awareness |
| **AI assistants** | ChatGPT, Claude | No persistence across sessions |
| **Calendar tools** | Reclaim.ai | Only time, not state |

**The gap:** No tool reconstructs the user's *current state* from raw, unstructured inputs.

## 2. Product Vision

### Three Pillars

| Pillar | Function |
|---|---|
| **Capture** | Text, images, PDFs, voice. Zero friction. |
| **State Engine (Cortex)** | AI extracts: action items, blockers, completions, deadlines, patterns |
| **Insight & Action** | Dashboard: what's happening, what needs attention, what's falling through |

### What Flowra Is NOT

| ❌ Not This | ✅ Instead |
|---|---|
| Knowledge base | Activity tracker |
| Document management | State inference |
| General AI chat | Accountability engine |
